\section{Implementations Overview}

\begin{frame}{Implementations}
  We introduced 5 different implementations of the Sciara-fv2 model on CUDA:
  \begin{itemize}
    \item \textbf{Naive Implementation}, by using global memory only.
    \item \textbf{Tiled implementation},  without using halo cells.
    \item \textbf{Tiled implementation}, with halo cells.
    \item \textbf{Cfame Implementation}.
    \item \textbf{Cfamo Implementation}, by memorizing outflows cells in shared memory.
  \end{itemize}
  
  \vspace{1em} % Spazio verticale
  Let's analyze them and how they perform.
\end{frame}
